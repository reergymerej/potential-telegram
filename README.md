This is a multi-player tic-tac-toe game using web sockets.  The first pass is
Node and vanilla JS on the client.

## install
```
yarn
```

## build & watch
```
yarn watch-client
```

## watch client source for dev
```
yarn watch-client
```

## start
```
./server/server.js
```

## open
```
http://localhost:1234
```
